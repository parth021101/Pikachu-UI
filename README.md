## Pokémon UI — Clone

A pixel-perfect frontend clone of a Pokémon Gen I showcase page, built with vanilla HTML and SCSS.

## Tech Stack
* HTML5
* SCSS (compiled to CSS)
* Bootstrap Icons
* Google Fonts — Google Sans Flex, Montserrat, Inter

## Features
* Split two-panel layout (yellow left + cream right)
* Large display typography with Google Sans Flex
* Pokémon character image overlapping both panels
* Faded Pokéball watermark decoration
* Dot navigation indicators
* Fully centered content with absolute positioning

## Tools & Assets
